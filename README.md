## 🚀 Quick Start

### Viewing Locally

1. Clone the repository:
```bash
git clone https://github.com/ahmed-raza-ur-rehman/ahmed-raza-ur-rehman.github.io.git
```

2. Navigate to the directory:
```bash
cd ahmed-raza-ur-rehman.github.io
```

3. Open in your browser:
```bash
# Using Python 3
python -m http.server 8000

# Or using Node.js
npx http-server

# Or simply open index.html in your browser
open index.html  # macOS
start index.html # Windows
xdg-open index.html # Linux
```

4. Visit `http://localhost:8000` in your browser

### Deployment

This site is automatically deployed via GitHub Pages. Any push to the `main` branch will trigger a deployment.

To deploy your own version:

1. Fork this repository
2. Go to Settings → Pages
3. Select the `main` branch as the source
4. (Optional) Add your custom domain in the CNAME file
5. Your site will be live at `https://your-username.github.io`

## 🎨 Customization

### Updating Content

1. **Personal Information**: Edit the text content in `index.html`
2. **Colors**: Modify CSS variables in the `:root` selector
3. **Projects**: Add/remove project cards in the Projects section
4. **Experience**: Update the timeline in the Experience section
5. **Skills**: Modify skill tags and categories in the Skills section

### Color Scheme

The site uses CSS variables for easy customization:

```css
:root {
    --bg-primary: #0d1117;
    --bg-secondary: #161b22;
    --text-primary: #e6edf3;
    --accent-primary: #58a6ff;
    /* ... more variables */
}
```

### Adding New Sections

1. Add the HTML section in `index.html`
2. Add corresponding styles in the `<style>` tag
3. Update navigation links in the navbar
4. Update `sitemap.xml` if needed
